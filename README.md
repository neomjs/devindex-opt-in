# DevIndex Opt-In

This repository serves a single purpose: providing an automated way for developers to opt-in or nominate others for the [Neo.mjs DevIndex](https://neo.mjs.com/apps/devindex/).

## How to Opt-In

There are two primary ways to be added to the DevIndex:

### Option 1: The "Quick Star" Method (For Yourself)

If you want to quickly add your own profile to the index, simply **Star this repository** at the top right of this page.

**What happens next?**
1. Our automated discovery pipeline runs hourly.
2. It detects your star, automatically verifying your GitHub identity.
3. Your GitHub username is added to our tracking queue.
4. Our updater pipeline evaluates your metrics. If your total contributions meet the minimum threshold, your profile will be indexed and displayed. If you do not yet meet the minimum threshold, you will be removed from the tracking queue. You are welcome to submit a new request once your contributions have grown!

**Note:** You can safely "Un-star" this repository after a few hours. The opt-in is permanent unless you later decide to [opt-out](https://github.com/neomjs/devindex-opt-out).

### Option 2: Issue Templates (For Yourself or Nominating Others)

If you prefer a documented request or want to nominate other developers you think should be featured:

1. Go to the **Issues** tab of this repository.
2. Click **New Issue**.
3. Choose either the **DevIndex Opt-In (Myself)** or **DevIndex Opt-In (Nominate Others)** template.
4. Fill out the form and submit.

Our automated pipeline will process these issues hourly, verify the usernames, and add them to our tracking queue.
